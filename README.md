# Cisco 2901 Router 3D Visualization

This project visualizes a Cisco 2901 router in photorealistic 3D using Three.js, GSAP, and Uzbek localization.

## Features
- Modular codebase (router sections)
- Uzbek port labels/tooltips
- GSAP animations
- UnrealBloomPass LED glow
- Exact rack dimensions

## Usage
Open `index.html` in your browser.

## Structure
- `src/` — Modular JS files
- `style.css` — Modern UI styling
- `index.html` — Entry point

## Requirements
- Three.js r128 (CDN)
- GSAP (CDN)
- Modern browser
